# Geyik

Geyik, sözcüklerin kökenine bakmanızı sağlayan küçük bir CLI uygulamasıdır.

## Kurulum

```bash
npm i -g geyik
geyik <sozcuk>
```

ya da direkt çalıştırmak için

```bash
npx geyik <sozcuk>
```

## Gereklilikler

- Node.js >= 18.17.1

## Kullanım

```

geyik <sozcuk>

```
